# AI-Booker — Студия производства аудиокниг на базе ИИ

> Превращаем книги в иммерсивные аудиоспектакли с актёрской игрой ИИ, атмосферным звуковым ландшафтом и профессиональным мастерингом — в один клик.

🌐 **Демо:** [booker-studio.lovable.app](https://booker-studio.lovable.app)

---

## Что такое AI-Booker?

AI-Booker — веб-приложение для автоматизированного производства аудиокниг в формате **радиоспектакля**. Глубокий семантический анализ текста, мульти-голосовой синтез речи, атмосферный саунд-дизайн и профессиональный DAW-таймлайн — с ИИ на каждом этапе.

### Ключевые возможности

| Возможность | Описание |
|-------------|----------|
| 📖 **Умный парсер** | Загрузка PDF → автоизвлечение оглавления → AI-сегментация на сцены с метаданными (настроение/темп) |
| 🎭 **Профайлер персонажей** | ИИ определяет персонажей, строит психологические портреты, автоматически подбирает голоса |
| 🎙️ **Мульти-провайдер TTS** | Yandex SpeechKit, SaluteSpeech (Sber), ElevenLabs, ProxyAPI/OpenAI — 40+ голосов с эмоциональным управлением |
| 🎬 **Раскадровка** | Слияние/разбиение блоков, паузы тишины, детекция инлайн-вставок, атрибуция спикеров |
| 🎵 **Движок атмосферы** | AI-генерация фоновых звуков, музыки и SFX (ElevenLabs) + интеграция с Freesound.org |
| 🎛️ **DAW-таймлайн** | Мультитрековое воспроизведение, микшер на каждый трек, мастер-цепочка (EQ→CMP→LIM→FLT→MBC→REV) |
| 🔌 **Per-Clip плагины** | Индивидуальные EQ, Compressor, Limiter, 3D Panner, Convolver (IR) на каждый клип |
| 🏛️ **Свёрточный ревербератор** | Библиотека IR (7 категорий), визуализация волны, прослушивание клипа через IR в реальном времени |
| 🎞️ **Оффлайн-рендер** | Сцена → 3 WAV-стема (Voice/Atmo/SFX) с per-clip FX через OfflineAudioContext |
| 🔊 **Про-мастеринг** | 5-полосный фильтр, 3-полосный MBC, стерео VU-метеринг, FFT-спектроанализатор с клик-курсором (частота + нота/октава) |
| 🤖 **AI-роли** | 6 специализированных ролей (Сценарист, Профайлер, Режиссёр и др.) с выделенными моделями |
| 💬 **AI-ассистент** | Контекстный чат-помощник, ведущий по процессу производства |
| 🎬 **Монтажная** | Финальная сборка главы: 3-трековый таймлайн стемов (Голос/Атмосфера/SFX), Trim/Fade с Undo/Redo, разбиение главы, стерео-редактор волноформ с клик-курсором для анализа частоты, полная мастер-цепочка |
| 📀 **Рендер главы** | Клиентский рендер через OfflineAudioContext с полной мастер-цепочкой, опциональная пиковая нормализация (-0.5 дБ), экспорт WAV (16/24/32-bit) и MP3 (128–320 kbps), прямое скачивание на компьютер пользователя (без загрузки в облако) |

## Пайплайн производства

```
PDF-файл
  │
  ▼
┌─────────────────────────────────┐
│  1. Загрузка + хранение         │  Парсер / Библиотека
│  2. Извлечение оглавления       │
│  3. Ручная правка структуры     │  Навигатор
│  4. Семантический анализ        │  AI (Сценарист) → границы сцен → метаданные
└─────────────┬───────────────────┘
              │  🎬 Передача главы
              ▼
┌─────────────────────────────────┐
│  5. Раскадровка (сегментация)   │  AI → типизация + фразы
│     5a. Редактирование блоков   │  Merge / Split / Delete + паузы тишины
│  6. Профилирование персонажей   │  AI (Профайлер) → психопортрет
│  7. Кастинг голосов             │  Автоподбор + ручная настройка
│  8. Предпрослушивание           │  Yandex / ElevenLabs / ProxyAPI
│  9. Синтез сцены                │  TTS → segment_audio → таймлайн
│  10. Таймлайн (Сцена/Глава)     │  Воспроизведение, seek, навигация
│  10a. Микшер                    │  Vol/Pan/Mute/Solo, Pre-FX, Reverb
│  10b. Мастеринг                 │  EQ→CMP→LIM→FLT→MBC→REV
│  10c. Метеринг                  │  Стерео VU L/R, FFT-спектр
│  10d. Per-Clip плагины          │  EQ→CMP→LIM→PAN3D→CONV
│  10e. Рендер в стемы            │  Voice/Atmo/SFX → WAV
└─────────────┬───────────────────┘
              │
              ▼
┌─────────────────────────────────┐
│  11. Атмосфера + SFX            │  Auto ✅ + Freesound ✅ + manual ✅
│  12. **Монтажная**                │  ✅ 3-трековый таймлайн, части главы, редактор, мастеринг
│  13. **Рендер главы**             │  ✅ Экспорт WAV/MP3, нормализация, скачивание на ПК
└─────────────────────────────────┘

🤖 Ассистент — доступен на любом этапе
🎭 AI Роли — специализированные модели для каждого типа задач
```

## Технический стек

| Слой | Технология |
|------|------------|
| Фронтенд | React, TypeScript, Vite, Tailwind CSS, shadcn/ui, Framer Motion |
| Аудио-движок | Tone.js + Web Audio API (OfflineAudioContext, Convolver, Panner3D, EQ, Compressor, Limiter, FFT) |
| PDF-обработка | pdfjs-dist (браузерная) |
| AI-анализ | Lovable AI Gateway (Gemini, GPT-5), ProxyAPI, OpenRouter |
| TTS | Yandex SpeechKit (v1/v3), SaluteSpeech (Sber), ElevenLabs, ProxyAPI/OpenAI TTS |
| Саунд-дизайн | ElevenLabs (SFX, Music), Freesound.org |
| Бэкенд | Lovable Cloud (PostgreSQL + Storage + Edge Functions) |
| Авторизация | Email + пароль, RLS-политики на все таблицы |

## Система AI-ролей

Каждая LLM-задача в пайплайне выполняется специализированной «ролью» со своим system prompt и оптимальной моделью:

| Роль | Зона ответственности | Tier |
|------|----------------------|------|
| 🌐 Переводчик | Перевод, i18n, локализация | lite |
| ✍️ Корректор | Ударения, SSML, пунктуация TTS | standard |
| 🎬 Сценарист | Сегментация текста, атрибуция спикеров | standard |
| 🎭 Режиссёр | Темп, паузы, эмоциональный рисунок | heavy |
| 🔍 Профайлер | Анализ персонажей, психотипы | heavy |
| 🎵 Звукоинженер | Промпты для SFX/атмосферы | standard |
